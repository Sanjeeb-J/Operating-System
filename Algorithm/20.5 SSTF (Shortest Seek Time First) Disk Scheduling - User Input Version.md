## **ALGORITHM: SSTF (Shortest Seek Time First) Disk Scheduling - User Input Version**

**Input:**
- Number of disk requests (n)
- Disk request sequence (cylinder numbers)
- Initial head position

**Output:**
- Total seek time using SSTF algorithm

**Steps:**

1. **START**
```c
#include <stdio.h>
#include <stdlib.h>
#include <limits.h>

int main()
```
1. **Initialize Variables:**
   - Declare `requests[100]` (array to store cylinder numbers)
   - Declare `visited[100] = {0}` (track processed requests, all initialized to 0)
   - Declare `n`, `head`, `totalSeek = 0`
```c
    int requests[100], visited[100] = {0};
    int n, head, totalSeek = 0;
```
1. **Input Number of Requests:**
   - **Print** "Enter number of requests: "
   - **Read** `n`
```c
    printf("Enter number of requests: ");
    scanf("%d", &n);
```
1. **Input Request Sequence:**
   - **Print** "Enter request sequence: "
   - Loop from `i = 0` to `i < n`
     - **Read** `requests[i]` (cylinder number for each request)
```c
    printf("Enter request sequence: ");
    for(int i = 0; i < n; i++)
        scanf("%d", &requests[i]);
```
1. **Input Initial Head Position:**
   - **Print** "Enter initial head position: "
   - **Read** `head`
```c
    printf("Enter initial head position: ");
    scanf("%d", &head);
```
1. **SSTF Disk Scheduling Algorithm:**
   - Loop from `i = 0` to `i < n`
     
     - **Declare** `minDist = INT_MAX` (initialize to maximum integer value)
     - **Declare** `index = -1` (no request selected yet)
     
     - **Find Closest Unvisited Request:**
       - Loop from `j = 0` to `j < n`
         - **If** `visited[j] == 0` **Then**
           - (Request not yet serviced)
           - `dist = abs(head - requests[j])` (calculate absolute distance)
           - **If** `dist < minDist` **Then**
             - (This request is closer than previous minimum)
             - `minDist = dist` (update minimum distance)
             - `index = j` (update selected request index)
     
     - **Service Selected Request:**
       - `visited[index] = 1` (mark request as serviced/visited)
       - `totalSeek += minDist` (add seek distance to total)
       - `head = requests[index]` (move head to this cylinder)
```c
for(int i = 0; i < n; i++) {
        int minDist = INT_MAX, index = -1;
        
        for(int j = 0; j < n; j++) {
            if(!visited[j]) {
                int dist = abs(head - requests[j]);
                if(dist < minDist) {
                    minDist = dist;
                    index = j;
                }
            }
        }
        
        visited[index] = 1;
        totalSeek += minDist;
        head = requests[index];
    }
```
1. **Output Results:**
   - **Print** "Total Seek Time (SSTF): " followed by `totalSeek`
```c
    printf("Total Seek Time (SSTF): %d\n", totalSeek);
```
1. **Return** 0

2. **END**
