## **ALGORITHM: Priority (Non-Preemptive) CPU Scheduling**

### **FUNCTION 1: findNextProcess()**

**Input:** Array of Process structures `p[]`, integer `n`, current `current_time`, and `is_completed[]` array

**Output:** Index of process with highest priority (lowest priority number) that has arrived and is not completed, or -1 if none available

**Steps:**

1. **START**
    
2. **Declare** variables:
    
    - `min_priority = INT_MAX` (initialize to maximum value)
    - `index = -1` (no process found yet)
3. **Find Process with Highest Priority:**
    
    - Loop from `i = 0` to `i < n`
        - **If** `p[i].arrival_time <= current_time` AND `is_completed[i] == 0` **Then**
            - (Process has arrived and is not yet completed)
                
            - **If** `p[i].priority < min_priority` **Then**
                
                - (Lower priority number = higher priority)
                - `min_priority = p[i].priority`
                - `index = i`
            - **Else If** `p[i].priority == min_priority` AND `p[i].arrival_time < p[index].arrival_time` **Then**
                
                - (Tie-breaker: choose process that arrived earlier)
                - `index = i`
4. **Return** `index`
    
5. **END**

### **FUNCTION 2: priorityNonPreemptiveScheduling()**

**Input:** Array of Process structures `p[]` and integer `n`

**Output:** Array with calculated completion_time, turnaround_time, and waiting_time; prints Gantt Chart

**Steps:**

1. **START**
    
2. **Declare** variables:
    
    - `current_time = 0` (current CPU time)
    - `completed = 0` (number of completed processes)
    - `is_completed[n]` (array to track completed processes)
3. **Initialize Completed Array:**
    
    - Loop from `i = 0` to `i < n`
        - `is_completed[i] = 0` (mark all processes as not completed)
4. **Print Gantt Chart Header:**
    
    - Print "Execution Order (Gantt Chart):"
5. **Priority Non-Preemptive Scheduling Loop:**
    
    - **While** `completed < n` **Do**
        - **Call** `findNextProcess(p, n, current_time, is_completed)` to get next process
            
        - Store result in `idx`
            
        - **If** `idx == -1` **Then**
            
            - (No process has arrived, CPU is idle)
            - `current_time++`
            - **Continue** to next iteration
        - **Print Execution:**
            
            - Print "Time current_time -> P[idx]"
        - **Execute Process (Non-Preemptive):**
            
            - `current_time += p[idx].burst_time`
            - (Execute entire process without interruption)
        - **Calculate Metrics:**
            
            - `p[idx].completion_time = current_time`
            - `p[idx].turnaround_time = p[idx].completion_time - p[idx].arrival_time`
            - `p[idx].waiting_time = p[idx].turnaround_time - p[idx].burst_time`
        - **Mark as Completed:**
            
            - `is_completed[idx] = 1`
            - `completed++`
6. **END**
    

---

### **FUNCTION 3: displayProcesses()**

**Input:** Array of Process structures `p[]` and integer `n`

**Output:** Table of all processes with their metrics and averages

**Steps:**

1. **START**
    
2. **Declare** variables:
    
    - `total_wt = 0` (total waiting time)
    - `total_tat = 0` (total turnaround time)
3. **Print Table Header:**
    
    - Print "Process | AT | BT | Priority | CT | TAT | WT"
4. **Print Each Process:**
    
    - Loop from `i = 0` to `i < n`
        - **Print** Process ID, Arrival Time, Burst Time, Priority, Completion Time, Turnaround Time, Waiting Time
        - **Add** waiting time to `total_wt`
        - **Add** turnaround time to `total_tat`
5. **Calculate and Print Averages:**
    
    - **Average Waiting Time** = `total_wt / n`
    - **Average Turnaround Time** = `total_tat / n`
6. **END**
    

---

### **MAIN FUNCTION: main()**

**Input:** Number of processes with their arrival time, burst time, and priority

**Output:** Priority scheduling table with averages and Gantt Chart

**Steps:**

1. **START**
    
2. **Input Number of Processes:**
    
    - **Read** `n`
3. **Declare** array of Process structures: `p[n]`
    
4. **Input Process Details:**
    
    - Loop from `i = 0` to `i < n`
        - `p[i].id = i + 1`
        - **Read** arrival time, burst time, and priority
        - Store in `p[i].arrival_time`, `p[i].burst_time`, and `p[i].priority`
5. **Perform Priority Non-Preemptive Scheduling:**
    
    - **Call** `priorityNonPreemptiveScheduling(p, n)`
6. **Display Results:**
    
    - **Call** `displayProcesses(p, n)`
7. **Return** 0
    
8. **END**

## **Key Concepts:**

|Term|Formula|Explanation|
|---|---|---|
|**Arrival Time (AT)**|Given|When process arrives in ready queue|
|**Burst Time (BT)**|Given|Total CPU time needed by process|
|**Priority (PR)**|Given|Priority number (lower = higher priority)|
|**Completion Time (CT)**|Previous CT + BT|When process finishes|
|**Turnaround Time (TAT)**|CT - AT|Total time from arrival to completion|
|**Waiting Time (WT)**|TAT - BT|Time spent waiting in ready queue|
|**Non-Preemptive**|No interruption|Once started, process runs to completion|
