```c
#include <stdio.h>
#include <unistd.h>
#include <pthread.h>
#include <semaphore.h>

#define READERS 3
#define WRITERS 2
#define OPERATIONS 5

sem_t resource;
sem_t readTry;
sem_t rmutex;
sem_t wmutex;

int readcount = 0;
int writecount = 0;
int dataValue = 0; // Shared data
```
---

## **ALGORITHM: Readers-Writers Problem (Writer-Preference Solution)**

### **FUNCTION 1: reader()**

**Input:** Thread argument containing reader ID

**Output:** Reader performs multiple read operations on shared data

**Steps:**

1. **START**
    
2. **Extract** reader ID from argument
    
3. **Loop** from `i = 0` to `i < OPERATIONS`
    
    - **ENTRY SECTION (Requesting Read):**
        - **Wait on Semaphore** `readTry`
            
            - `sem_wait(&readTry)`
            - (Allows writers to block new readers)
        - **Wait on Semaphore** `rmutex` (reader mutex)
            
            - `sem_wait(&rmutex)`
            - (Protect readcount from race condition)
        - **Increment** `readcount++` (one more reader)
            
        - **If** `readcount == 1` **Then**
            
            - (First reader, block all writers)
            - **Wait on Semaphore** `resource`
                - `sem_wait(&resource)`
        - **Signal Semaphore** `rmutex`
            
            - `sem_post(&rmutex)`
        - **Signal Semaphore** `readTry`
            
            - `sem_post(&readTry)`
            - (Allow other readers to proceed)
    - **CRITICAL SECTION (Read Operation):**
        - **Print** "Reader [id] reading value: [dataValue]"
        - **Sleep** for 1 second
    - **EXIT SECTION (Finishing Read):**
        - **Wait on Semaphore** `rmutex`
            
            - `sem_wait(&rmutex)`
        - **Decrement** `readcount--` (one less reader)
            
        - **If** `readcount == 0` **Then**
            
            - (Last reader, allow writers)
            - **Signal Semaphore** `resource`
                - `sem_post(&resource)`
        - **Signal Semaphore** `rmutex`
            
            - `sem_post(&rmutex)`
    - **Sleep** for 1 second
4. **Exit Thread:** `pthread_exit(NULL)`
    
5. **END**
    
```c
void *reader(void *arg)
{
  int id = *(int *)arg;

  for (int i = 0; i < OPERATIONS; i++)
  {

    // ENTRY SECTION
    sem_wait(&readTry);
    sem_wait(&rmutex);

    readcount++;
    if (readcount == 1)
      sem_wait(&resource);

    sem_post(&rmutex);
    sem_post(&readTry);

    // CRITICAL SECTION
    printf("Reader %d reading value: %d\n", id, dataValue);
    sleep(1);

    // EXIT SECTION
    sem_wait(&rmutex);
    readcount--;
    if (readcount == 0)
      sem_post(&resource);
    sem_post(&rmutex);

    sleep(1);
  }

  pthread_exit(NULL);
}
```

### **FUNCTION 2: writer()**

**Input:** Thread argument containing writer ID

**Output:** Writer performs multiple write operations on shared data

**Steps:**

1. **START**
    
2. **Extract** writer ID from argument
    
3. **Loop** from `i = 0` to `i < OPERATIONS`
    
    - **ENTRY SECTION (Requesting Write):**
        - **Wait on Semaphore** `wmutex` (writer mutex)
            
            - `sem_wait(&wmutex)`
            - (Protect writecount from race condition)
        - **Increment** `writecount++` (one more writer waiting)
            
        - **If** `writecount == 1` **Then**
            
            - (First writer, block all new readers)
            - **Wait on Semaphore** `readTry`
                - `sem_wait(&readTry)`
                - (Prevents new readers from entering)
        - **Signal Semaphore** `wmutex`
            
            - `sem_post(&wmutex)`
        - **Wait on Semaphore** `resource` (exclusive access)
            
            - `sem_wait(&resource)`
            - (Wait for current readers/writers to finish)
    - **CRITICAL SECTION (Write Operation):**
        - **Increment** `dataValue++` (modify shared data)
        - **Print** "Writer [id] writing value: [dataValue]"
        - **Sleep** for 2 seconds (simulate write time)
    - **EXIT SECTION (Finishing Write):**
        - **Signal Semaphore** `resource`
            
            - `sem_post(&resource)`
            - (Release exclusive access)
        - **Wait on Semaphore** `wmutex`
            
            - `sem_wait(&wmutex)`
        - **Decrement** `writecount--` (one less writer)
            
        - **If** `writecount == 0` **Then**
            
            - (Last writer, allow new readers)
            - **Signal Semaphore** `readTry`
                - `sem_post(&readTry)`
                - (Allow readers to enter)
        - **Signal Semaphore** `wmutex`
            
            - `sem_post(&wmutex)`
    - **Sleep** for 1 second
4. **Exit Thread:** `pthread_exit(NULL)`
    
5. **END**
    
```c
void *writer(void *arg)
{
  int id = *(int *)arg;

  for (int i = 0; i < OPERATIONS; i++)
  {

    // ENTRY SECTION
    sem_wait(&wmutex);
    writecount++;
    if (writecount == 1)
      sem_wait(&readTry); // Block readers
    sem_post(&wmutex);

    sem_wait(&resource);

    // CRITICAL SECTION
    dataValue++;
    printf("Writer %d writing value: %d\n", id, dataValue);
    sleep(2);

    sem_post(&resource);

    // EXIT SECTION
    sem_wait(&wmutex);
    writecount--;
    if (writecount == 0)
      sem_post(&readTry); // Allow readers
    sem_post(&wmutex);

    sleep(1);
  }

  pthread_exit(NULL);
}
```

### **MAIN FUNCTION: main()**

**Input:** None

**Output:** Synchronized read-write operations with writer preference

**Steps:**

1. **START**
    
2. **Declare** variables:
    
    - `r[READERS]` (array of reader threads)
    - `w[WRITERS]` (array of writer threads)
    - `rid[READERS]` (array of reader IDs)
    - `wid[WRITERS]` (array of writer IDs)
3. **Initialize Semaphores:**
    
    - **Call** `sem_init(&resource, 0, 1)` (binary semaphore for resource access, initial value = 1)
    - **Call** `sem_init(&readTry, 0, 1)` (binary semaphore to block readers, initial value = 1)
    - **Call** `sem_init(&rmutex, 0, 1)` (binary semaphore for readcount protection, initial value = 1)
    - **Call** `sem_init(&wmutex, 0, 1)` (binary semaphore for writecount protection, initial value = 1)
4. **Create Reader Threads:**
    
    - Loop from `i = 0` to `i < READERS`
        - `rid[i] = i + 1` (assign reader ID)
        - **Call** `pthread_create(&r[i], NULL, reader, &rid[i])`
5. **Create Writer Threads:**
    
    - Loop from `i = 0` to `i < WRITERS`
        - `wid[i] = i + 1` (assign writer ID)
        - **Call** `pthread_create(&w[i], NULL, writer, &wid[i])`
6. **Wait for All Reader Threads:**
    
    - Loop from `i = 0` to `i < READERS`
        - **Call** `pthread_join(r[i], NULL)`
7. **Wait for All Writer Threads:**
    
    - Loop from `i = 0` to `i < WRITERS`
        - **Call** `pthread_join(w[i], NULL)`
8. **Destroy Semaphores:**
    
    - **Call** `sem_destroy(&resource)`
    - **Call** `sem_destroy(&readTry)`
    - **Call** `sem_destroy(&rmutex)`
    - **Call** `sem_destroy(&wmutex)`
9. **Return** 0
    
10. **END**
    
```c
int main()
{

  pthread_t r[READERS], w[WRITERS];
  int rid[READERS], wid[WRITERS];

  // Initialize semaphores
  sem_init(&resource, 0, 1);
  sem_init(&readTry, 0, 1);
  sem_init(&rmutex, 0, 1);
  sem_init(&wmutex, 0, 1);

  // Create reader threads
  for (int i = 0; i < READERS; i++)
  {
    rid[i] = i + 1;
    pthread_create(&r[i], NULL, reader, &rid[i]);
  }

  // Create writer threads
  for (int i = 0; i < WRITERS; i++)
  {
    wid[i] = i + 1;
    pthread_create(&w[i], NULL, writer, &wid[i]);
  }

  // Join threads
  for (int i = 0; i < READERS; i++)
    pthread_join(r[i], NULL);

  for (int i = 0; i < WRITERS; i++)
    pthread_join(w[i], NULL);

  // Destroy semaphores
  sem_destroy(&resource);
  sem_destroy(&readTry);
  sem_destroy(&rmutex);
  sem_destroy(&wmutex);

  return 0;
}
```

## **Key Concepts:**

|Concept|Purpose|
|---|---|
|**resource**|Semaphore controlling exclusive access to shared data|
|**readTry**|Semaphore that writers use to block new readers|
|**rmutex**|Protects `readcount` variable|
|**wmutex**|Protects `writecount` variable|
|**readcount**|Number of active readers|
|**writecount**|Number of waiting/active writers|
|**Writer-Preference**|Writers get priority over new readers|
|**Mutual Exclusion**|Only one writer at a time, readers blocked during write|

---

## **Semaphore Summary:**

|Semaphore|Initial|Protected Variable|Purpose|
|---|---|---|---|
|`resource`|1|Shared data|Exclusive access to data|
|`readTry`|1|Reader entry|Writers block new readers|
|`rmutex`|1|readcount|Protect readcount updates|
|`wmutex`|1|writecount|Protect writecount updates|

---

## **Synchronization Rules (Writer-Preference):**

```
READERS:
1. Wait on readTry (check if writers are waiting)
2. If first reader, acquire resource (block writers)
3. Multiple readers can read simultaneously
4. If last reader, release resource (allow writers)

WRITERS:
1. Increment writecount (register as waiting writer)
2. If first writer, acquire readTry (block new readers)
3. Acquire resource (wait for current readers)
4. Write data exclusively
5. Release resource
6. If last writer, release readTry (allow new readers)
```
