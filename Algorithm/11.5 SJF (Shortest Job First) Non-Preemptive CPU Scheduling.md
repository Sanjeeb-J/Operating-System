## **ALGORITHM: SJF (Shortest Job First) Non-Preemptive CPU Scheduling**

### **FUNCTION 1: findNextProcess()**

**Input:** Array of Process structures `p[]`, integer `n`, current `current_time`, and `completed[]` array

**Output:** Index of process with shortest burst time (that has arrived and not completed), or -1 if none available

**Steps:**

1. **START**
    
2. **Declare** variables:
    
    - `min_time = INT_MAX` (initialize to maximum value)
    - `index = -1` (no process found yet)
3. **Find Process with Minimum Burst Time:**
    
    - Loop from `i = 0` to `i < n`
        - **If** `p[i].arrival_time <= current_time` AND `completed[i] == 0` **Then**
            - (Process has arrived and is not yet completed)
                
            - **If** `p[i].burst_time < min_time` **Then**
                
                - `min_time = p[i].burst_time`
                - `index = i`
            - **Else If** `p[i].burst_time == min_time` AND `p[i].arrival_time < p[index].arrival_time` **Then**
                
                - (Tie-breaker: choose process that arrived earlier)
                - `index = i`
4. **Return** `index`
    
5. **END**
    

---

### **FUNCTION 2: sjfNonPreemptive()**

**Input:** Array of Process structures `p[]` and integer `n`

**Output:** Array with calculated completion_time, turnaround_time, and waiting_time

**Steps:**

1. **START**
    
2. **Declare** variables:
    
    - `current_time = 0` (current CPU time)
    - `completed_count = 0` (number of completed processes)
    - `completed[n]` (array to track completed processes)
3. **Initialize Completed Array:**
    
    - Loop from `i = 0` to `i < n`
        - `completed[i] = 0` (mark all processes as not completed)
4. **SJF Scheduling Loop:**
    
    - **While** `completed_count < n` **Do**
        - **Call** `findNextProcess(p, n, current_time, completed)` to get next process
            
        - Store result in `index`
            
        - **If** `index == -1` **Then**
            
            - (No process has arrived, CPU is idle)
            - `current_time++`
            - **Continue** to next iteration
        - **Execute Process (Non-Preemptive):**
            
            - `current_time += p[index].burst_time`
            - (Execute entire process without interruption)
        - **Calculate Metrics:**
            
            - `p[index].completion_time = current_time`
            - `p[index].turnaround_time = p[index].completion_time - p[index].arrival_time`
            - `p[index].waiting_time = p[index].turnaround_time - p[index].burst_time`
        - **Mark as Completed:**
            
            - `completed[index] = 1`
            - `completed_count++`
5. **END**
    

---

### **FUNCTION 3: displayProcesses()**

**Input:** Array of Process structures `p[]` and integer `n`

**Output:** Table of all processes with their metrics and averages

**Steps:**

1. **START**
    
2. **Declare** variables:
    
    - `ttt = 0` (total turnaround time)
    - `twt = 0` (total waiting time)
3. **Print Table Header:**
    
    - Print "PID | AT | BT | CT | TAT | WT"
4. **Print Each Process:**
    
    - Loop from `i = 0` to `i < n`
        - **Print** Process ID, Arrival Time, Burst Time, Completion Time, Turnaround Time, Waiting Time
        - **Add** turnaround time to `ttt`
        - **Add** waiting time to `twt`
5. **Calculate and Print Averages:**
    
    - **Average Turnaround Time** = `ttt / n`
    - **Average Waiting Time** = `twt / n`
6. **END**
    

---

### **MAIN FUNCTION: main()**

**Input:** Number of processes and their arrival times and burst times

**Output:** SJF scheduling table with averages

**Steps:**

1. **START**
    
2. **Input Number of Processes:**
    
    - **Read** `n`
3. **Declare** array of Process structures: `p[n]`
    
4. **Input Process Details:**
    
    - Loop from `i = 0` to `i < n`
        - `p[i].id = i + 1`
        - **Read** arrival time and burst time
        - Store in `p[i].arrival_time` and `p[i].burst_time`
5. **Perform SJF Non-Preemptive Scheduling:**
    
    - **Call** `sjfNonPreemptive(p, n)`
6. **Display Results:**
    
    - **Call** `displayProcesses(p, n)`
7. **Return** 0
    
8. **END**

## **Key Concepts:**

|Term|Formula|Explanation|
|---|---|---|
|**Arrival Time (AT)**|Given|When process arrives in ready queue|
|**Burst Time (BT)**|Given|Total CPU time needed by process|
|**Completion Time (CT)**|Previous CT + BT|When process finishes|
|**Turnaround Time (TAT)**|CT - AT|Total time from arrival to completion|
|**Waiting Time (WT)**|TAT - BT|Time spent waiting in ready queue|
|**SJF (Non-Preemptive)**|Shortest Job First|Process with shortest burst time executes fully|
|**Non-Preemptive**|No interruption|Once started, process runs to completion|
